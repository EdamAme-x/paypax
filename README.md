### Installation

- Node.js or Bun
```shell
npm i --save-dev paypay.x.js
or
yarn add -D paypay.x.js
or
pnpm i -save-dev paypay.x.js
or
bun i -D paypay.x.js
```

- Deno
```typescript
import PayPay from "https://deno.land/x/paypayjs/mod.ts"
...
```
